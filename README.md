how 2 use

get a smithing template (/ascendance give template <quantity> <tier> <blessed> <bias>)

use it with a netherite ingot on any item

win (up to 3 times)


blessed: prevents CoV addition

bias: reroll up to 2 more times so you are more likely to get the bias enchantment in question


note: you need to use the universal_netherite.zip datapack to permit things other than diamond; by default using the templates will result in the template being returned, the plugin will block the recipe
